Workshop Question 6: Recursion - Fibomulti

Please read important information first.

Description

First, get the file path with input() and do not use any prompt messages. You are supposed to read an integer `n` from a file. Then, you are expected to write a recursive function under the definition of `fibomulti` given to you. A fibomulti number is the multiple of its previous 2 fibomulti numbers. First fibomulti number: 2; Second fibomulti number: 3 (e.g. fibomulti(1):=2, fibomulti(2):=3). The `fibomulti` function eventually returns the `n`th fibomulti number. It is like fibonacci, but you should use multiplication instead of addition.
You can ignore the count variable. It will be printed automatically.

Your function is expected to:
1) count the number of calls to itself and keep this value in count variable.
2) return the nth fibomulti number.

In the last editable block (DO_NOT_EDIT part), call your function as res = fibomulti(n) and print the result as shown below.

Note: Using recursion is mandatory; other solutions are not accepted.
