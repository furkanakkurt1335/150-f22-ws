Workshop Question 6: Recursion - Fibomulti

Please read important information first.

Description

You are supposed to read an integer `n` from a file. Then, you are expected to write a recursive function under the definition of `fibomulti` given to you. A fibomulti number is the multiple of its previous 2 fibomulti numbers. First fibomulti number: 2. Second fibomulti number: 3. The `fibomulti` function eventually returns the `n`th fibomulti number. It is like fibonacci, but you should use multiplication instead of addition.
You can ignore the count variable. It will be printed automatically.

Note: Using recursion is mandatory; other solutions are not accepted.
Input Output
file1.txt 
Result: 18
Count: 5
file2.txt 
Result: 408146688
Count: 41
file3.txt 
Result: 1944
Count: 15
