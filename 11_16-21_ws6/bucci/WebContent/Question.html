Workshop Question 5: Recursion - Bucci

Please read important information first.

Description

You are supposed read an integer `n` from a file. Then, you are expected to write a recursive function under the definition of `bucci` given to you. A Bucci number is the sum of its previous 2 Bucci numbers. First Bucci number: 4. Second Bucci number: 7. The `bucci` function eventually returns the `n`th Bucci number. Note: Using recursion is mandatory; other solutions are not accepted.

file1.txt

Count: 57313
