Workshop Question 6: Recursion - Bucci

Please read important information first.

Description

First, get the file path with input() and do not use any prompt messages. You are supposed to read an integer `n` from a file. Then, you are expected to write a recursive function under the definition of `bucci` given to you. A Bucci number is the sum of its previous 2 Bucci numbers. First Bucci number: 4. Second Bucci number: 7. (e.g. bucci(1):=4, bucci(2):=7) The `bucci` function eventually returns the `n`th Bucci number.

Your function is expected to:
1) count the number of calls to itself and keep this value in count variable.
2) return the nth bucci number.

In the last editable block (DO_NOT_EDIT part), call your function as res = bucci(n) and print the result as shown below. Note: Using recursion is mandatory; other solutions are not accepted.
