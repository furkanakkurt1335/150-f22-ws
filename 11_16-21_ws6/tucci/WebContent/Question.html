Workshop Question 6: Recursion - Tucci

Please read important information first.

Description

You are supposed to read an integer `n` from a file. Then, you are expected to write a recursive function under the definition of `tucci` given to you. A Tucci number is the result of the previous Tucci number's division by the second previous Tucci number. First Tucci number: 15. Second Tucci number: 29. (e.g. third Tucci number is 29/15) The `tucci` function eventually returns the `n`th Tucci number.

Your function is expected to:
1) count the number of calls to itself and keep this value in count variable.
2) return the nth tucci number.

In the last editable block (DO_NOT_EDIT part), call your function as res = tucci(n) and print the result as shown below.

Note: Using recursion is mandatory; other solutions are not accepted.
Input Output
file1.txt 
Result: 15.00000
Count: 25
file2.txt 
Result: 0.06667
Count: 5
file3.txt 
Result: 1.93333
Count: 21891
