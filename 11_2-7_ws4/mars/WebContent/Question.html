Workshop Question 4: Mathematical Operators with File I/O

Please read important information first.

Description

You will be given 2 files as input in order in the forms of 'file*.txt' and 'integer*.txt' (* represents a number). You are supposed to read the first file ('file*.txt') entirely, convert the content of the file to an integer and store it as a variable. This integer represents how many characters you are supposed to read from the second file ('integer*.txt'). You are supposed to convert the content of the second file (read only as much as the stored variable amount) to an integer if it's not given as such (possibly an alphabetic character). This integer represents seconds. You are requested to print a string, according to the following instructions:

1. If the integer in seconds represents less than or equal to 1 day, print "Less than 1 day".

2. If the integer in seconds represents more than 1 day, print "More than 1 day".

3. If the integer in seconds represents more than 1 year, print "More than 1 year". Do not print "More than 1 day".

Years and days are of Mars, 687 days and 25 hours, respectively. In the end, only 1 line will be printed.

Input Output
file1.txt and integer1.txt 
More than 1 day
file2.txt and integer2.txt 
Less than 1 day
file3.txt and integer3.txt 
More than 1 year
