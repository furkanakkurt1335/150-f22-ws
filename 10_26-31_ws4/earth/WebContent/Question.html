Workshop Question 4: Mathematical Operators with File I/O

Please read important information first.

Description

You will be given 2 files as input in order in the forms of 'file*.txt' and 'integer*.txt' (* represent a number). You are supposed to read the first file ('file*.txt') entirely, convert the content of the file to an integer and store it as a variable. This integer represents how many characters you are supposed to read from the second file ('integer*.txt'). You are supposed to convert the content of the second file (read only as much as the stored variable amount) to an integer if it's not given as such (possibly an alphabetic character). This integer represents seconds. You are requested to print strings, according to the following instructions, in separate lines (using '\n'):

1. If the integer in seconds represents more than 1 year, print "More than 1 year"

2. If the integer in seconds represents more than 1 day, print "More than 1 day"

Years and days are of Earth, 365 days and 24 hours, respectively.

Input Output
file1.txt and integer1.txt 
More than 1 day
file2.txt and integer2.txt 
More than 1 day
file3.txt and integer3.txt 
More than 1 year
More than 1 day
