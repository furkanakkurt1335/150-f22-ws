Description

We will use a cylinder which has a radius r and a height h , for this question. We will also use a cup and a glass.

Read the four variables r, h, cup_volume, glass_volume from a given file name, each placed in a separate line, respectively.

Write three functions calculate_cylinder_base_area, calculate_cylinder_lateral_area, and calculate_cylinder_volume.

calculate_cylinder_base_area must calculate the cylinder's base area (circle) as 3 * r2 and return that value, taking the PI as 3.

calculate_cylinder_lateral_area must calculate 2 * 3 * r * h and return that value, taking the PI as 3.

calculate_cylinder_volume must evaluate the volume of the cylinder by 3 * r 2 * h .

We have a cup which is larger than our cylinder and we wish to fill it by using cylinder. Display how many times we should use the cylinder (which has its base removed) to fill the cup fully. (The cup's volume is a multiple of the cylinder's volume.)

Next, we have a glass which is smaller than our cylinder and we wish to fill the cylinder (which has its base removed) by using the glass's entire volume; however this time, the cylinder's volume is not a multiple of the glass so the cylinder will not be full in the end. Display as integer what percentage of the cylinder's volume will be empty.
