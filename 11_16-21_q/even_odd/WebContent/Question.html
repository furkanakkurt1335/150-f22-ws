Workshop Question 6: Recursion - Even or Odd

Please read important information first.

Description

First, get the file path with input() and do not use any prompt messages. You are supposed to read two integers `n` (2 digit number) and `t` from a file where n specifies the number and t denotes the turn. Then, you are expected to write a recursive function under the definition of `even_odd` given to you. The `even_odd` function must return the `t`'th Even_odd number of `n`. The first even_odd number (at t=1) is the reverse of `n` if `n` is even, 11 more if `n` is odd. Even_odd numbers will go on until they become a three digit number. Note: Using recursion is mandatory; other solutions are not accepted. The following table is just for illustration purposes. You are not supposed to write intermediate results on the screen. Input: n=13, t=5
5 13
4 24
3 42
2 24
1 42
Res: 24
