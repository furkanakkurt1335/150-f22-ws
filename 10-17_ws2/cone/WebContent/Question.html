Description

We will use a cone which has a radius r and a height h , for this question. We will also use a cup and a glass.

Read the four variables r, h, cup_volume, glass_volume from a given file name, each placed in a separate line, respectively.

Write three functions calculate_cone_base_area, calculate_cone_lateral_area, and calculate_cone_volume.

calculate_cone_base_area must calculate the cone's base area (circle) as 3 * r2 and return that value, taking the PI as 3.

calculate_cone_lateral_area must calculate 3 * r * √ h 2 * r 2 and return that value, taking the PI as 3.

calculate_cone_volume must evaluate the volume of the cone by 3 * r 2 * h / 3 .

We have a cup which is larger than our cone and we wish to fill it by using this cup. Display how many times we should use the cone (which has its base removed) to fill the cup fully. (The cup's volume is a multiple of the cone's volume.)

Next, we have a glass which is smaller than our cone and we wish to fill the cone (which has its base removed) by using the glass's entire volume; however this time, the cone's volume is not a multiple of the glass so the cone will not be full in the end. Display as integer what percentage of the cone's volume will be empty.
