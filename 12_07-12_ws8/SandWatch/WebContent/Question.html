Workshop Question 8: Sand Watch

Please read important information first.

Description

In this question, you are asked to print a Sand Clock

You need to read two integers from the file as the first number (f) and second number (s), respectively.
Check if any of the numbers are negative. If so, display INVALID INPUT and call `exit()`.

Generate a single random integer between the first integer from the input and the second integer from the input. Take that random integer as the count of star in the first output line.
Print the sand clock just like the ones in the examples. Output the result in an individual line.
DO NOT write any prompt messages inside the input functions. Be careful with your output, such as spacing in the output or upper case messages.
