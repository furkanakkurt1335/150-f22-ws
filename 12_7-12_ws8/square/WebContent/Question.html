Workshop Question 8: Square

Please read important information first.

Description

In this question, you are asked to draw a square with the print function.

You need to read two integers from a file as the first number (f) and second number (s), respectively.
Check if any of the numbers are negative. If so, display INVALID INPUT and call `exit()`.

Generate a single random integer between the first integer from the input and the second integer from the input. Take that random integer as the count of star in the side of the square.
Print the square just like the ones in the examples.
DO NOT write any prompt messages inside the input functions. Be careful with your output, such as spacing unnecessarily.
