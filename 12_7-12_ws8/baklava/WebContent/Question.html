Workshop Question 8: Baklava

Please read important information first.

Description

In this question, you are asked to print a diamond-shaped Baklava slice.

You need to read two integers from a file as the first number (f) and second number (s), respectively.
Check if any of the numbers are negative. If so, display INVALID INPUT and call `exit()`.

Generate a single random integer between the first integer from the input and the second integer from the input. Take that random integer as the count of star in the middle of the shape(which is the largest part of it).
Print the baklava slice just like the ones in the examples. Output the result in an individual line.
DO NOT write any prompt messages inside the input functions. Be careful with your output, such as spacing in the output or upper case messages.
